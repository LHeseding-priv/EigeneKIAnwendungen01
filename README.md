# EigeneKIAnwendungen01

Kleines Beispielprojekt mit Notebooks und Datensätzen zur Arbeit mit TensorFlow und scikit-learn.

## Inhalt
- `iris01.ipynb`: Beispiel-Notebook zur Klassifikation des Iris-Datensatzes.
- `docs/`: Dokumentation und zusätzliche Beispiele (wird nicht versioniert — siehe `.gitignore`).

## Schnellstart
1. Virtuelle Umgebung erstellen (empfohlen):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt  # optional, falls vorhanden
```

2. Jupyter-Notebook starten und `iris01.ipynb` öffnen:

```bash
jupyter notebook iris01.ipynb
```

3. Notebook-Zellen ausführen (Reihenfolge beachten).

## Hinweise
- Das Verzeichnis `docs/` ist in `.gitignore` aufgeführt und wird nicht ins Repository übernommen.
- Falls Sie das Projekt in ein Git-Repository aufnehmen möchten, führen Sie `git init` und einen ersten Commit durch.
